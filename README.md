# LogoType Lab

A lightweight browser tool for designing experimental typography logos with Type, Blur, Pixel, Slice, and Line effects.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy

This project is static and can be deployed directly to Vercel. The social sharing thumbnail is defined in `og-image.png` and wired through the Open Graph and Twitter meta tags in `index.html`.
